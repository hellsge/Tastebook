# ☕ 程序员咖啡外卖评价手册

Programmer's Guide to Coffee Delivery Reviews

一个轻量的咖啡外卖记录仓库，帮你追踪喝过的每一杯咖啡，找到最值得复购的那杯。

## 怎么用

- 打开 `index.html` 即可录入新记录，自动复制 CSV 行到剪贴板
- 数据存储在 `coffee.csv`，纯文本，随时可编辑
- 支持按价格、评分、单星价格排序，快速找到性价比之王

## 数据字段

| 字段 | 说明 |
|------|------|
| 品名 | 咖啡名称 |
| 品牌 | 店铺/品牌 |
| 价格 | 实付金额（¥） |
| 评分 | ⭐ 1-5 星 |
| 单星价格 | 价格 ÷ 评分，越低越值 |
| 评价 | 简短口感描述 |
| 备注 | 可选，冰/热、糖度、加料等 |

## 评分标准

| 评分 | 含义 |
|------|------|
| ⭐ | 垃圾还钱 |
| ⭐⭐ | 一言难尽 |
| ⭐⭐⭐ | 无功无过 |
| ⭐⭐⭐⭐ | 值得一试 |
| ⭐⭐⭐⭐⭐ | 惊为天人 |

## 贡献

欢迎 PR 你喝过的咖啡记录，格式参考 `coffee.csv` 现有条目即可。
